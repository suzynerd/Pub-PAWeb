Task: Cliente Define o Projeto a ser Desenvolvido
A definição do projeto ocorre através de uma entrevista com o cliente, onde o mesmo expõe
o problema que deseja solucionar. Se o problema for entendido pela equipe de desenvolvimento,
passa-se para a definição do escopo, senão entrevista-se novamente o cliente, até que se tenha
entendimento do problema.
