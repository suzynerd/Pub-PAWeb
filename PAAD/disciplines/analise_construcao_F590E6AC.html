Main Description

As atividades consitem basicamente em analisar o caso de uso, identificando conceitos, operações e componentes; atividades que podem acontecer em paralelo.

Em seguida o Diagrama de Interação de Objetos (UML) para o (s) Caso (s) de Uso (s) Implementado (s) é construído e o Diagrama de Classes de Projeto (UML) atualizado. É importante salientar que é nessa disciplina que os componentes são identificados. Eles serão a base para a implementação, que ocorrerá orientada a componentes.
